easyBanner

let you create web banner element easily and quickly.
options download github

Options

The options below are you can custom.

Variable Type Default Description
animation String "slide" 轮播动画的模式，有3种方案可选：["slide", "fade", "none"] （"none"代表没有任何动画效果）
trigger String "click" 通过序列元素来触发动画的事件类型，可以使用"click"或"hover"
arrow Boolean true 显示左右方向箭头
serial Boolean
String		 true 序列元素，可选参数为[true, false, "equal", "thumb"]，true为默认的圆点按钮，"equal"为等宽的矩形，"thumb"为开启缩略图功能
autoPlay Boolean true 开启自动轮播
speed Int 600 轮播动画的速度
interval Int 5000 自动轮播每次动画的间隔
during Function $.noop 在动画进行时执行的回调函数，该函数可以有两个形参[$this, currentIndex], $this为轮播容器的元素，currentIndex为轮播当前显示项的索引
after Function $.noop 在动画完成时执行的回调函数，该函数可以有两个形参[$this, currentIndex], $this为轮播容器的元素，currentIndex为轮播当前显示项的索引
