Lazyload

Use the "data-src" attribute to open the function of lazyload

  • HTML
  • CSS
  • JAVASCRIPT

<div class="banner-equal">
    <ul>
        <li><img data-src="img/banner-1.jpg"></li> // use data-src to replace src
        <li><img data-src="img/banner-2.jpg"></li>
        <li><img data-src="img/banner-3.jpg"></li>
        <li><img data-src="img/banner-4.jpg"></li>
        <li><img data-src="img/banner-5.jpg"></li>
    </ul>
</div>


.banner-automatic{
    height: 625px; //  must give the banner height
    border: 5px solid #fff;
    border-radius: 5px;
}


$('.banner-lazyload').easyBanner();
