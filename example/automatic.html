easyBanner

let you create web banner element easily and quickly.
options download github

Automatic

Add the necessary elements and style automatically, it doesn't matter even less wrote some style.

  • HTML
  • CSS
  • JAVASCRIPT

<div class="banner-automatic">
    <ul>
        <li><img src="img/banner-1.jpg"></li>
        <li><img src="img/banner-2.jpg"></li>
        <li><img src="img/banner-3.jpg"></li>
        <li><img src="img/banner-4.jpg"></li>
        <li><img src="img/banner-5.jpg"></li>
    </ul>
</div>


.banner-automatic{
    height: 625px; //  must give the banner height
    border: 5px solid #fff;
    border-radius: 5px;
}


$('.banner-automatic').easyBanner();

Animation - Fade

easyBanner support three kinds of animation: slide(default), fade, none(no animation)

  • HTML
  • CSS
  • JAVASCRIPT

<div class="banner-fade">
    <div> // can use any tag here
        <a><img src="img/banner-1.jpg"></a>  // you can use any tag as list item
        <a><img src="img/banner-2.jpg"></a>
        <a><img src="img/banner-3.jpg"></a>
        <a><img src="img/banner-4.jpg"></a>
        <a><img src="img/banner-5.jpg"></a>
    </div>
</div>


.banner-fade{
    height: 625px;
    border: 5px solid #fff;
    border-radius: 5px;
}


$('.banner-fade').easyBanner({
    animation: 'fade'
});

Custom Style

You can also define you own page style.

  • HTML
  • CSS
  • JAVASCRIPT

<div class="banner-custom">
    <ul> // can use any tag here
        <li><img src="img/banner-6.jpg"></li>
        <li><img src="img/banner-7.jpg"></li>
        <li><img src="img/banner-8.jpg"></li>
        <li><img src="img/banner-9.jpg"></li>
        <li><img src="img/banner-10.jpg"></li>
    </ul>
</div>


.banner-custom .eb-arrow{width: 90%;}  // default width is 96%
.banner-custom .eb-arrow a{position: relative;width: 64px;height: 64px;opacity: 0;cursor: pointer;}
.banner-custom .eb-arrow a.prev{left: -50px;background: url(img/icon-prev.png);}
.banner-custom .eb-arrow a.next{right: -50px;background: url(img/icon-next.png);}
.banner-custom:hover .eb-arrow a.prev{left: 0;opacity: 1;}
.banner-custom:hover .eb-arrow a.next{right: 0;opacity: 1;}

.banner-custom .eb-serial{bottom: 15px;}
.banner-custom .eb-serial a{width: 8px;height: 8px;background: #eee;border-radius: 5px;}
.banner-custom .eb-serial a.active{width: 15px;background: #00b579;transition: width .3s;-webkit-transition: width .3s;}


$('.banner-custom').easyBanner();

Serial - Equal

let the width of the serial buttons equal.

  • grape
  • raspberry
  • strawberry
  • blueberry
  • cherry
  • HTML
  • CSS
  • JAVASCRIPT

<div class="banner-serial">
    <ul>
        <li><img src="img/banner-1.jpg"><span>grape</span></li>
        <li><img src="img/banner-2.jpg"><span> raspberry</span></li>
        <li><img src="img/banner-3.jpg"><span>strawberry</span></li>
        <li><img src="img/banner-4.jpg"><span>blueberry</span></li>
        <li><img src="img/banner-5.jpg"><span>cherry</span></li>
    </ul>
</div>


.banner-serial .eb-list ul li{position: relative;}
.banner-serial .eb-list ul li span{position: absolute;bottom: 10px;left: 0;width: 100%;background: rgba(0, 0, 0, .5);font: 26px/50px "Arial";color: #fff;text-indent: 1em;letter-spacing: 1px;}
.banner-serial .eb-serial{bottom: 0;}
.banner-serial .eb-serial a{background: #00b579;} // default color is #fff
.banner-serial .eb-serial a.active{background: #ffa500;} // default color is #ffa500


$('.banner-serial').easyBanner({
    serial: 'equal',
    after: function($this, currentIndex) {
        $('.eb-list ul li span', $this).hide();
        $('.eb-list ul li span', $this).eq(currentIndex).show(function() {});
    }
});
